Metodología

Transparencia sobre
la transparencia

Todo nuestro código es open source. Cada clasificación incluye su razonamiento completo. El algoritmo es auditable por cualquier ciudadano.

1
Extracción de datos
Ingestamos el BOE diariamente vía su API oficial, los indicadores del INE, los programas electorales de todos los partidos con representación parlamentaria y las memorias ministeriales.
BOE API INE API Apache Airflow Scrapy
2
Análisis con IA
Cada promesa electoral se vectoriza semánticamente y se compara contra el corpus legal. Un modelo de lenguaje evalúa la evidencia y clasifica el estado con una cadena de razonamiento explícita.
sentence-transformers pgvector LLM análisis
3
Auditoría humana
Un Comité de Metodología formado por periodistas de datos, académicos y representantes de la sociedad civil revisa y valida cada actualización metodológica antes de publicarse.
Open source (AGPL) GitHub público Changelog auditable
